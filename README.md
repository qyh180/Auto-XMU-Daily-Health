# Auto-XMU-Daily-Health
用于学习交流的厦门大学每日健康打卡脚本
### 声明：该项目仅用于学习和交流技术，请同学们按照自己实际情况打卡，造成的一切后果本人概不负责

首先感谢原作者的代码，在此基础上添加了以下功能：

1. 获取已打卡日期。
2. 将打卡情况通过邮件发送，内容包括：打卡成功\失败，已打卡xx天。
